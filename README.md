## Dependencies

### Install npm via nodejs using either

- [NodeJS](https://nodejs.org/en/)
- [NVM](https://github.com/nvm-sh/nvm)

### Install dependencies

```sh
yarn install
```

### Running the application

```sh
yarn dev
```
